Класс Ability необходим для определения способностей (заклинаний) героя. 

Наследуется от интерфейса INotifyPropertyChanged для создания MVVM-модели представения данных (используется дополнительный класс ViewModelAbilities). 

Имеет приватные поля: 
string _name; 
int _manaCost; 
int _level;
int _damage;
которые связаны с соответствующими автосвойствами данного класса, и которые нужны для работы MVVM-модели.

Автосвойства данного класса:
string Name - название способности (заклинания)
int ManaCost - необходимое кол-во маны на использование способности (заклинания)
int Level - уровень, необходимый для изучения способности
int Damage - значение наносимого урона 

Имеет конструктор по умолчанию и с параметрами.
В конструктор с параметрами передаются: string name, int manaCost, int level, int damage. Заполняются соответствующие поля.

Методы:
Объявлен int-овый метод AttackAbility, который в качестве параметра принимает аргумент типа Ability. 
Возвращает величину наносимого способностью героя урона.

Объявлен void-овский метод OnPropertyChanged. Неоходим для корректной работы MVVM-модели.
