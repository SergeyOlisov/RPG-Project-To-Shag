Абстрактный класс Editor необходим для создания, загрузки и сохранения переменных героя и врага. 

Методы:
Статический метод типа Enemy EnemyCreate, который в качестве параметров принимает string name, int level, int health, int mana, List<Ability> abilities.
Возвращает переменную типа Enemy.

Статический метод типа Hero HeroCreate, без аргументов.
Возвращает переменную типа Hero.

Статический метод void EnemySerialize, который в качестве параметра принимает переменную типа Enemy.
Асихронно сериализует данные типа Enemy в json-файл. 

Статический метод void HeroSerialize, который в качестве параметра принимает переменную типа Hero и путь для сохранения string path.
Асихронно сериализует данные типа Hero в json-файл. 

Статический метод void EnemyDeserialize, который в качестве параметра принимает переменную string path.
Асихронно возвращает десериализованные данные типа Enemy. 

Статический метод void HeroDeserialize, который в качестве параметра принимает переменную string path.
Асихронно возвращает десериализованные данные типа Hero.

Статический метод void AttackSpellSerialize, который в качестве параметра принимает переменную типа Ability.
Асихронно сериализует данные типа Ability в json-файл. 

Статический метод Ability AttackSpellDeserialize, который в качестве параметра принимает переменную string path.
Асихронно возвращает десериализованные данные типа Ability.

Статический метод void AttackSpellsSerialize, который в качестве параметра принимает список типа Ability.
Асихронно сериализует список данных типа Ability в json-файл с установленным путём по умолчанию.
