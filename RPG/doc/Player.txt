Статический класс Player необходим для объявления в начале игры экземпляра класса Hero статическим, а также описывает ряд общих полей и методов.  

Имеет приватное поле const string _tempHeroPath - путь к дефолтному json-файлу с параметрами героя.

Имеет публичные статические поля:
Hero Hero - экземпляр класса Hero становится статическим.
int HealthPotions - количество бутылочек зелий здоровья
int ManaPotions - количество бутылочек зелий маны


Методы:
Метод void UseHealthPotion, без аргументов. 
Отвечает за использование зелий здоровья.

Метод void UseManaPotion, без аргументов. 
Отвечает за использование зелий маны.

Метод типа Hero CreatePlayer(), без аргументов. 
Возвращает переменную с экземляром класса Hero.

Метод типа Hero LoadingPlayer(), без аргументов. 
Возвращает переменную типа Hero путём десериализации данных из txt- или json-файлов. Путь выбирается пользователем вручную.

Метод типа Hero HeroSave(), без аргументов. 
Сохраняет данные типа Hero путём сериализации их в txt- или json-файлы. Путь выбирается пользователем вручную.

Метод типа Hero QuickLoad(), без аргументов. 
Возвращает переменную типа Hero путём десериализации данных из txt- или json-файлов. Путь установлен по умолчанию.

Метод типа Hero QuickSave(), без аргументов. 
Сохраняет данные типа Hero путём сериализации их в txt- или json-файлы. Путь установлен по умолчанию.
