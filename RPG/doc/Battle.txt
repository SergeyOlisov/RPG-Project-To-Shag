Абстрактный класс Battle необходим для определения методов логики боя. 

Методы:
Статический метод void HeroAttack, который в качестве параметра принимает ссылку на переменную типа Enemy.
Уменьшает величину здоровья врага на величину физической атаки героя и проверяет врага на смерть методом IsEnemyDead.

Статический метод void HeroSpellCaste, который в качестве параметра принимает ссылку на переменную типа Enemy и аргумент типа Ability.
Уменьшает величину здоровья врага на величину урона применённой способности героя (заклинания) и проверяет врага на смерть методом IsEnemyDead.

Статический метод void EnemyAttack, который в качестве параметра принимает ссылку на переменную типа Enemy.
Проверяется возможность уклонения героя от атаки врага (рандомно, но разброс в зависимости от ловкости героя).
При неуклонении уменьшает величину здоровья героя на величину физической атаки врага и проверяет врага на смерть методом IsHeroDead.

Статический метод void IsEnemyDead, который в качестве параметра принимает ссылку на переменную типа Enemy.
Проверяет количество здоровья врага и при значении <= 0, выводит сообщение и производит вызов функции LevelUp, производящей возможное увеличение уровня героя.

Статический метод void IsEnemyDead, без аргументов.
Проверяет количество здоровья героя и при значении <= 0, выводит сообщение.
