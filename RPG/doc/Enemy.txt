Класс Enemy определяет характеристики врага.

Имеет поля:
string Name - имя врага
int Level - уровень врага
int Experience - сколько опыта дают за убийство врага
int Health - уровень здоровья врага
int Mana - уровень маны врага
List<Ability> Ability - список способностей врага

Класс имеет конструктор по умолчанию, а так же конструктор с параметрами.
В конструктор с параметрами передаётся аргумент типа Hero, т.к в зависимости от уровня героя зависит уровень врага и его характирестики. 
Второй конструктор с параметрами (// для редактора) пока не используется.

Методы:

Метод int Damage возвращает показатель урона с рандомным разбросом.

Метод int Dead возвращает количество получаемого игроком опыта при убийстве врага.

Метод void LevelUpEnemy вызывает метод UpdateEnemy.

Метод void UpdateEnemy повышает характеристики врага.
